# template-vue-component

> SAO vue standalone component template

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao vue-component my-project
```

### From git

```bash
sao sevody/template-vue-component my-project
```
