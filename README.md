# template-vue-component

> SAO vue standalone component template

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From git

```bash
$ sao sevody/template-vue-component my-component
$ cd my-component
$ npm install
$ npm run serve
```

### Linting

```bash
$ npm run lint
```

### Testing & Test Coverage

```bash
$ npm run test
$ npm run test:watch
$ npm run test:cov
```

### Building

#### ES6 Modules / CommonJS

```bash
$ npm run build
```

#### UMD

```bash
$ npm run build:umd
```
